Wizard Chat

Link to live project: Wizard Chat

Technologies used: HTML, CSS, JavaScript, PHP, MySQL

About

This project was made as a way to play Dungeons and Dragons. While I was making it I:
  - Built my own account creation/login system.
  - Built a UI to display the following:
    - Options for speech to text.
    - Ability to select different LLM's.
    - A taggable advance setting section that give access to the AI's
      memory and system prompt.
    - A chat history list with options to delete the past chats you don't want.
    - A chat display that shows options to resend last sent chat and delete
      past messages sent.
    - A chat input area that grows as the message that is being typed out grows.
    - A toggle mic button.
  - Worked with third party API's that were hosting the LLM's
  - Built a system to store, retrieve, and update all the chats for each user.
  - Built a custom long term memory system.
  - Built a credit system to keep users from spamming messages and draining my bank account.
  - Integrated a paypal so people could buy more credits.
  - Built a home page that displays different D&D games you can play through.
  - Built a system so you can make your own games.

Dice Calculator

Link to live project: Dice Calculator

Link to github repo: Dice Calculator

Technologies used: HTML, CSS, JavaScript, PHP

About

This was a small project that I made to help me learn php. The user inputs the number of dice (x) that
they want to roll, then "d" for die, then the size of the dice (y), is "xdy" format. Example 1d20. If
the user wants to add a modifier at the end of the roll, they can just add it on, example: 2d6+5. The
string with all of the information is then sent to the server where php code parses the string for the
information and then returns the dice roll to the user.
