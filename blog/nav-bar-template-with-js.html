How to make a nav bar template using JavaScript.

Web frameworks have a way to take and make a template that allows to reuse html code in multiple webpages, if you are not using a framework most people will tell you that you should use one to take and template your nav bar. Although for a lot of projects that might be correct, if you are making a static website, there should be no need to take and use framework just for making templates.

Take a look at this code here. This code is the code that I had in every page of my website when I was first making it. I was getting tired of every time I made a new page, or I wanted to make some changes to it, having to take and change the code in multiple files. 

            
<nav>
    <a href="index.html"> <img src="images/logo.png" class="logo" alt="Logo" ></img></a>
    <ul>
        <li><a href="index.html">Home</a></li>
        <li><a href="projects.html">Projects</a></li>
        <li><a href="about.html">About</a></li>
        <li><a href="blog.html">Blog</a></li>
    </ul>
</nav>

Step 1: Create the JavaScript File

Make a file and place it in the same directory as your html files. I named my JavaScript file "navBar.js".

Step 2: Use the document.write Method

The document.write method in JavaScript allows you to take whatever is inside of it and put it on the page. An important thing to note about this is that when it used while the page is still loading, it slots the text inside of it in to the html of the page like it is more html. If it is used after the page has finished loading, it will take and replace all of the code on the page with what is inside of it.

Take and write the following into your JavaScript file that you made in step one. 

            
document.write(

)

You will notice that I left a space in the code, I did this so that I can put in the html is a easy to read way.

Step 3: Put in Your HTML Header Code

You are going to want put in your code line by line, like it is a array. Around each line you need to put quotation marks and after each line you need to put a comma. Also, don't forget to put backslashes before every quotation mark in your HTML Code so that it shows on the page and does not mess up your code. See the code below as a example. 

            
document.write(
    '<nav>',
        '<a href=\"index.html\"> <img src=\"images/logo.png\" class=\"logo\" alt=\"Logo\" ></img></a>',
        '<ul>',
            '<li><a href=\"index.html\">Home</a></li>',
            '<li><a href=\"projects.html\">Projects</a></li>',
            '<li><a href=\"about.html\">About</a></li>',
            '<li><a href=\"blog.html\">Blog</a></li>',
        '</ul>',
    '</nav>'
)

Step 4: Implementing the Code

Now that you have the JavaScript code that you need, the next step is to reference the file as the site is loading, you do this by taking and just referencing the JavaScript file in the right place. In this case, the right place inside of the header element of the HTML files. A example is down below 

            
<header>
    <script src="navBar.js"></script>
</header>

If you run this code now, you should see it working, bellow I will walk through some modification that I made to the code to make it work on my website.

Bonus Step 1: Letting the code know what page it was on.

In my nav bar, depending which page it is on, I wanted to have a way to take change the color of the active page as shown picture below.

You will notice in the picture above that the blog page, which is the current active page is highlighted. This done by putting a active class after the li element in the code.

The first thing to solve here is how to get info on which page it is on, I did this by takeing and making the JavaScript into a function as shown bellow. 

            
function navBarDisplay() {
    document.write(
        "<nav>",
            `<a href=\"index.html\"> <img src=\"images/logo.png\" class=\"logo\" alt=\"Logo\" ></img> </a>`,
                "<ul>",
                        `<li><a href=\"index.html\">Home</a></li>`,
                        `<li><a href=\"projects.html\">Projects</a></li>`,
                        `<li><a href=\"about.html\">About</a></li>`,
                        `<li><a href=\"blog.html\">Blog</a></li>`,
                "</ul>",
        "</nav>"
    )
}

Next this code need a extra step to display the code in the html. 

            
<header>
    <script src="navBar.js"></script>
    <script>navBarDisplay()</script>
</header>
            
        
            
function navBarDisplay(page) {
    let indexClass = ""
    if (page == "index") {
        indexClass = "class=\"active\""
    } else {
        indexClass = ""
    }
    if (page == "projects") {
        projectClass = "class=\"active\""
    } else {
        projectClass = ""
    }
    if (page == "about") {
        aboutClass = "class=\"active\""
    } else {
        aboutClass = ""
    }
    if (page == "blog") {
        blogClass = "class=\"active\""
    } else {
        blogClass = ""
    }
    document.write(
        "<nav>",
            `<a href=\"index.html\"> <img src=\"images/logo.png\" class=\"logo\" alt=\"Logo\" ></img> </a>`,
                "<ul>",
                        `<li ${indexClass}><a href=\"index.html\">Home</a></li>`,
                        `<li ${projectClass}><a href=\"projects.html\">Projects</a></li>`,
                        `<li ${aboutClass}><a href=\"about.html\">About</a></li>`,
                        `<li ${blogClass}><a href=\"blog.html\">Blog</a></li>`,
                "</ul>",
        "</nav>"
    )
}
